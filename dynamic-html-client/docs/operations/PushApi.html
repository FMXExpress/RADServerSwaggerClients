URL

http://localhost:8080/push

HTTP Method

POST

Response Type

Parameters

  • Body: body PushObject(PushObject)

    Object containing the Push Message data structure

  • Header: X-Embarcadero-Application-Id String

  • Header: X-Embarcadero-App-Secret String

  • Header: X-Embarcadero-Master-Secret String
