This is the API

deleteResourceEndpoint

Used to invoke the DELETE method of the resource from an existing EMS EdgeModule.

URL

http://localhost:8080/edgemodules/{mname}/{rname}

HTTP Method

DELETE

Response Type

Parameters

  • Path: mname String

    Is the EMS EdgeModule name

  • Path: rname String

    Is the unique EMS EdgeModule Resource name

  • Header: X-Embarcadero-Application-Id String

  • Header: X-Embarcadero-App-Secret String

  • Header: X-Embarcadero-Master-Secret String
