This is the API

addUser

| Used to add a new `User` object to the EMS database.

URL

http://localhost:8080/users

HTTP Method

POST

Response Type

useridObject

Parameters

  • Body: body UserCredentialsObject(UserCredentialsObject)

    Object to add a new EMS User in the EMS Server

  • Header: X-Embarcadero-Application-Id String

  • Header: X-Embarcadero-App-Secret String

  • Header: X-Embarcadero-Master-Secret String

getUsers

| Used to retrieve all data from `Users`. Optional query param of **skip, limit, order, where** determines size of returned array, filtered and order by.

URL

http://localhost:8080/users

HTTP Method

GET

Response Type

List

Parameters

  • Header: X-Embarcadero-Application-Id String

  • Header: X-Embarcadero-App-Secret String

  • Header: X-Embarcadero-Master-Secret String

  • Query: skip BigDecimal

    users skipped

  • Query: limit BigDecimal

    maximum number of results to return

  • Query: order BigDecimal

    order ascending or descending (asc, desc)

  • Query: where String

    filter operators (lt, lte, gt, gte, eq, neq, like, nlike)
