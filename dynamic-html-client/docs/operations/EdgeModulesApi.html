This is the API

getModule

| Used to retrieve all data from `EdgeModule`. Optional query param of **skip, limit, order, where** determines size of returned array, filtered and order by.

URL

http://localhost:8080/edgemodules/{mname}

HTTP Method

GET

Response Type

edgeModuleObject

Parameters

  • Path: mname String

    Is the unique EMS EdgeModule identifier

  • Header: X-Embarcadero-Application-Id String

  • Header: X-Embarcadero-App-Secret String

  • Header: X-Embarcadero-Master-Secret String

getModuleResource

| Used to retrieve all data from a resource of an EdgeModule. **mname** is the unique EMS EdgeModule identifier in the EMS database. **resourcename** is the name of the resource.

URL

http://localhost:8080/edgemodules/{mname}/resources/{name}

HTTP Method

GET

Response Type

edgeModuleResourceObject

Parameters

  • Path: mname String

    Is the unique EMS EdgeModule identifier

  • Path: name String

    Is the unique EMS EdgeModule Resource name

  • Header: X-Embarcadero-Application-Id String

  • Header: X-Embarcadero-App-Secret String

  • Header: X-Embarcadero-Master-Secret String
