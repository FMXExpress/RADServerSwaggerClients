This is the API

addInstallation

| Used to add a new `Installation` object to the EMS database.

URL

http://localhost:8080/installations

HTTP Method

POST

Response Type

installationAddedObject

Parameters

  • Body: body InstallationAddObject(InstallationAddObject)

    Object to sign up a new EMS User in the EMS Server

  • Header: X-Embarcadero-Application-Id String

  • Header: X-Embarcadero-App-Secret String

  • Header: X-Embarcadero-Master-Secret String

deleteInstallation

| Used to delete an `Installation`. **id** is the unique EMS Installation identifier (IID) in the EMS database.

URL

http://localhost:8080/installations/{id}

HTTP Method

DELETE

Response Type

Parameters

  • Path: id String

    A Installation ID

  • Header: X-Embarcadero-Application-Id String

  • Header: X-Embarcadero-App-Secret String

  • Header: X-Embarcadero-Master-Secret String

getChannels

http://localhost:8080/installations/channels

GET

Response Type

List

Parameters

  • Header: X-Embarcadero-App-Secret String

  • Header: X-Embarcadero-Master-Secret String

GET

Response Type

Parameters

  • Path: id String

    A Installation ID

  • Header: X-Embarcadero-Application-Id String

  • Header: X-Embarcadero-App-Secret String

  • Header: X-Embarcadero-Master-Secret String

  • Header: X-Embarcadero-Master-Secret String

getInstallations

| Used to retrieve all data from `Installation`. Optional query param of **skip, limit, order, where** determines size of returned array, filtered and order by.

URL

http://localhost:8080/installations

HTTP Method

GET

Response Type

List

Parameters

  • Header: X-Embarcadero-Application-Id String

  • Header: X-Embarcadero-App-Secret String

  • Header: X-Embarcadero-Master-Secret String

  • Query: skip BigDecimal

    users skipped

  • Query: limit BigDecimal

    maximum number of results to return

  • Query: order BigDecimal

    order ascending or descending (asc, desc)

  • Query: where String

    filter operators (lt, lte, gt, gte, eq, neq, like, nlike)

updateInstallation

| Used to update an `Installation`. **id** is the unique EMS Installation identifier (IID) in the EMS database.

URL

http://localhost:8080/installations/{id}

HTTP Method

PUT

Response Type

updatedInstallationObject

Parameters

  • Path: id String

    A Installation ID

  • Body: body UpdateInstallationObject(UpdateInstallationObject)

    Installation fields to update

  • Header: X-Embarcadero-Application-Id String

  • Header: X-Embarcadero-App-Secret String

  • Header: X-Embarcadero-Master-Secret String
