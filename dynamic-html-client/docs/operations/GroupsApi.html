This is the API

addGroup

| Used to add a new `Group` object to the EMS database.

URL

http://localhost:8080/groups

HTTP Method

POST

Response Type

Parameters

  • Body: body GroupAddObject(GroupAddObject)

    Object to sign up a new EMS User in the EMS Server

  • Header: X-Embarcadero-Application-Id String

  • Header: X-Embarcadero-App-Secret String

  • Header: X-Embarcadero-Master-Secret String

deleteGroup

| Used to delete a `Group`. **item** is the unique EMS Group name in the EMS database.

URL

http://localhost:8080/groups/{item}

HTTP Method

DELETE

Response Type

Parameters

  • Path: item String

    A group name

  • Header: X-Embarcadero-Application-Id String

  • Header: X-Embarcadero-App-Secret String

  • Header: X-Embarcadero-Master-Secret String

getGroup

| Used to retrieve all data from a `Group`. **item** is the unique EMS Group name in the EMS database.

URL

http://localhost:8080/groups/{item}

HTTP Method

GET

Response Type

groupObject

Parameters

  • Path: item String

    A group name

  • Header: X-Embarcadero-Application-Id String

  • Header: X-Embarcadero-App-Secret String

  • Header: X-Embarcadero-Master-Secret String

getGroupFields

| Used to retrieve all the `field names` of the EMS Group (including the custom fields).

URL

http://localhost:8080/groups/fields

HTTP Method

GET

Response Type

List

Parameters

  • Header: X-Embarcadero-Application-Id String

  • Header: X-Embarcadero-App-Secret String

  • Header: X-Embarcadero-Master-Secret String

getGroups

| Used to retrieve all data from `Groups`. Optional query param of **skip, limit, order, where** determines size of returned array, filtered and order by.

URL

http://localhost:8080/groups

HTTP Method

GET

Response Type

List

Parameters

  • Header: X-Embarcadero-Application-Id String

  • Header: X-Embarcadero-App-Secret String

  • Header: X-Embarcadero-Master-Secret String

  • Query: skip BigDecimal

    users skipped

  • Query: limit BigDecimal

    maximum number of results to return

  • Query: order BigDecimal

    order ascending or descending (asc, desc)

  • Query: where String

    filter operators (lt, lte, gt, gte, eq, neq, like, nlike)

updateGroup

| Used to update a `Group`. **item** is the unique EMS Group name in the EMS database.

URL

http://localhost:8080/groups/{item}

HTTP Method

PUT

Response Type

updatedGroupObject

Parameters

  • Path: item String

    A group name

  • Body: body UpdateGroupObject(UpdateGroupObject)

    Object to sign up a new EMS User in the EMS Server

  • Header: X-Embarcadero-Application-Id String

  • Header: X-Embarcadero-App-Secret String

  • Header: X-Embarcadero-Master-Secret String
